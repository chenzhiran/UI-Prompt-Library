Product Roadmap

从零到开发者：
你的 CodeCombat 进阶蓝图

CodeCombat为你设计了一条清晰、可追踪的成人学习路径，将游戏关卡转化为扎实的工程能力。

01

语法学徒

Python / JavaScript

Core

在游戏关卡中掌握变量、循环、函数、条件判断等核心语法。

Project

编写算法控制英雄移动、攻击、收集资源。

建立坚实的编程直觉，能独立解决数百个微型逻辑问题。

02

全栈见习生

Web Development

Core

学习HTML/CSS，并运用Python/JavaScript开发交互式网页游戏。

Project

构建自己的“克隆”版经典游戏（如2048、贪吃蛇），并部署上线。

理解前端与后端的基础交互，形成完整的“产品”开发视角。

03

框架工匠

React, Node.js 等

Core

在更复杂的游戏项目中，引入主流框架和工具链，管理复杂状态和组件。

Project

开发多人在线策略游戏的界面或后台服务模块。

体验现代化、工程化的协作开发流程，为真实工作环境做好准备。

04

开源贡献者

Open Source Workflow

Core

深入学习Git与开源工作流，直接为CodeCombat的游戏引擎或关卡库提交代码。

Project

修复Bug、设计新关卡、开发新功能。

构建你的可验证贡献记录，这是你技术简历上最闪亮的一笔。

每一步，你都在编写真实的代码。

构建真实的项目，从今天开始。
